How Worgle Determines the Daily Word

1

Set the Epoch

The algorithm uses June 19, 2021 as the starting epoch date.

2

Calculate Days Since Epoch

Compute the number of days between the target date and the epoch.

days_since_epoch = (target_date - epoch_date)
3

Compute the Solution Index

Subtract the offset of 207, then take modulo of 986 (solution list length).

index = (days_since_epoch - 207) % 986
4

Look Up the Word

Use the computed index to pick the word from the 986-word solution list.

word = solutions[index]
